# 梦幻西游 AI 自动辅助 使用说明

## 一、系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 |
| Python | 3.11 或 3.12（不要用 3.14） |
| 显卡 | NVIDIA GPU（可选，训练 YOLO 用） |
| 游戏 | 梦幻西游 PC 版，800×600 窗口 |

## 二、快速开始

```bash
setup_deps.bat        # 安装依赖
launcher.bat          # 带控制台启动
launcher.vbs          # 无控制台启动
```

首次运行会自动创建 `data/`、`yolo_dataset/`、`debug/` 目录。

## 四、任务功能

| 任务 | 说明 |
|------|------|
| 师门(20轮) | F8回师门→模板找师傅→OCR识别类型→执行→交 |
| 捉鬼(10轮) | 钟馗→天眼→找鬼→战斗→交 |
| 剧情(通用) | 红字+OCR定位追踪栏→螺旋修正点击→自动寻路→循环 |
| 巡逻 | 循环走动+遇怪自动战斗，适合挂机 |
| 押镖 | 接镖→护送→交镖 |

## 五、核心技术

| 层级 | 技术 |
|------|------|
| 鼠标 | SendInput(ctypes) + 贝塞尔曲线 + dead zone + 验证重试 |
| 视觉 | YOLOv8 + 感知哈希 + 模板匹配 + 像素diff + 颜色检测 |
| 窗口 | HWND_TOPMOST + WindowFromPoint + 点击位置验证 |
| 战斗 | 6方法共识弹窗检测 + 角色分类 + 智能选目标 |

## 六、YOLO 模型训练

已内置 20 张训练数据。主面板 → 训练 YOLO → 开始训练。

## 七、AI 助手（可选）

支持 OpenAI / Anthropic / Ollama。主面板 → AI 设置 → 选提供商 → 填 Key → 保存。

## 八、云端存储

http://119.91.225.128:9527 — 上传/下载数据集和模型。

## 九、常见问题

| 问题 | 解决 |
|------|------|
| 启动闪退 | Python 3.12，debug_run.bat 诊断 |
| 未找到窗口 | 游戏打开后点"重新定位窗口" |
| 点错被踢 | 弹窗选人保护已内置，采集弹窗模板更佳 |
| 无模型 | 点"训练YOLO"用内置数据训练 |
| 换电脑 | 云端存储→下载模型和数据集 |
